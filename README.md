#  Space Shooter Game

A simple 2D arcade shooter game built using **Python** and **Pygame**.

##  Features
- Player spaceship movement
- Enemy ships spawning
- Laser shooting system
- Collision detection
- Score tracking system
- Smooth gameplay using Pygame

##  Controls

| Key | Action |
|-----|------|
| ⬅️ | Move Left |
| ➡️ | Move Right |
| ⬆️ | Move Up |
| ⬇️ | Move Down |
| Space | Shoot Laser |

## 🛠 Tech Stack
- Python
- Pygame
